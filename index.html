Hi, I'm Colin! In addition to this website, here are some things I've made.

Idea Webber

Idea Webber is a browser-based tool for mapping concepts and connecting ideas. It's also the biggest web app I've ever worked on. It's primarily designed for desktop, though it works on mobile as well.

Made with HTML, CSS, Javascript, Firebase Database, and Firebase Storage.

Small Tools

For skill practice or fun, I've made a few small web app tools. For example, I've made a search engine for Wikipedia articles, a calculator based on the Windows 10 calculator app, and a search engine for SpongeBob SquarePants quotes.

Made with HTML, CSS, Javascript, and Unity (C#).

Video Games

I've created and released around 22 video games which you can find on my itch.io page. Most of them are playable on any desktop broswer.

Made with Unity (C#) and GameMaker Studio (GML).
